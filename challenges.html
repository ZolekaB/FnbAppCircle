1. HTML Structure Challenge

Create a valid HTML document with all the essential elements.

Challenge: Your document must include:
  • DOCTYPE declaration
  • HTML, head, and body tags
  • A title in the head section
  • At least one heading and one paragraph in the body

2. Headings Challenge

Use all six heading levels to create a document outline.

Challenge: Create a page with:
  • All heading levels from h1 to h6
  • Appropriate content for each heading level

3. Image Gallery Challenge

Create a photo gallery with multiple images.

Challenge: Requirements:
  • At least 3 images
  • Consistent sizing (use width or height attributes)
  • A heading for your gallery

You can use these test image URLs: https://picsum.photos/200, https://picsum.photos/300, https://picsum.photos/400

4. Multi-page Website Challenge

Create a simple multi-page website with navigation.

Challenge: Create:
  • A home page (index.html)
  • An about page
  • A contact page
  • Navigation links between all pages

Practice this in our Interactive Playground or in your local development environment.
